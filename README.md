Smart Data Pipeline Project

This project demonstrates a real-time big data pipeline architecture using Apache ecosystem tools.

Architecture Diagram

Project Workflow
Data Source generates streaming data.
Apache Kafka handles data ingestion.
Spark Streaming processes incoming data.
Data is stored in HDFS Data Lake.
Apache Spark performs batch processing.
Hive provides querying and presentation layer.
Power BI visualizes insights and analytics.
Airflow orchestrates workflow scheduling.
